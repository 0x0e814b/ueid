# ueid

Unique Emoji Identifier

```ts
import { ueidv4 } from 'ueid';
const ueid = ueidv4(); // 🚃🌼🍏🚨🔗🌳🌷🔗🌇😚🔗🍋🎋🏡🍒🌄🍋🎃
```

```html
<script src="ueid.js"></script>
<script>
  console.log(ueid.ueidv4()) // 🏃🌻🎅🍮🍃🔗🚒🚱🔗🚻🚄🔗🚑🏒🚚🚞🍻🚧🌺
</script>

```
